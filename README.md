# Hover Notes
- Userscript for adding notes on the go, while you browse or use the internet.
- Slide-in notes panel appears on screen when you hover the bottom-left edge, disappears on inactivity.
- Per-site notes. Stores notes for upto 2 days (locally, of course).
- Exportability – use export option that appears under Hover Panel in Userscript Extension

## How to Use
- Install Tampermonkey or Violentmonkey or another Userscript Extension of choice
- Create new userscript 
- Paste code from 'Hover Notes' file in this repository
